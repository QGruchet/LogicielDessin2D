# Description

Ceci est un logiciel de dessin 2D. Ici seul l'affichage textuel est présent. 

# Utilisation
L'utilisateur a plusieur choix de commande :
	-dessin : commande de dessin du programme qui contient plusieurs commande et qui donne toutes les infos sur les figures dessiné:
		-- triangle : dessine un traingle à l'aide des coordonnées de points entrés par l'utilisateur;
		-- rectangle : dessine un traingle à l'aide des coordonnées de points entrés par l'utilisateur;
		-- cercle : dessine un traingle à l'aide du rayon et du point du centre entré par l'utilisateur;
		-- carre : dessine un carre à l'aide des coordonnées du point et la longueur entrés par l'utilisateur;
	-bouger : permet de bouger les figures à l'aide des points entrés par l'utilisateur;
	-vider : permet de vider le tableaux contenant toutes les figures;
	-help : contient l'ensembles des commandes disponibles ainsi qu'un léger descriptif;


# Compilation et execution
Pour compiler ce programme deux solutions:
- Depuis votre editeur, appuyer sur le bouton "build" puis "run main"
- Depuis un terminal entrez les commandes suivantes :
	-javac main.java
	-java main
